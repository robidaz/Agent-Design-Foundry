# KnowledgeX — RFP / RFI Response Agent

## Overview

KnowledgeX is the proof of concept for **Task 2 — RFP/RFI Response Agent**.

Beyond the design spec ([KnowledgeX.md](KnowledgeX.md)), I built a working agent that performs hybrid retrieval over a curated 8-document knowledge base. The vector database (pgvector) runs locally via Docker Compose, and embeddings are generated by a local sentence-transformers model (`BAAI/bge-large-en-v1.5`) running on my Ubuntu desktop — no third-party embedding API in the loop. Reasoning, coverage judgment, and answer drafting are then performed by Claude via the Anthropic API.

The [output](output) folder contains results from two engagement runs (`ENG-000` and `ENG-001`), with retrieval and scoring thresholds tuned between them. I halted iteration once the second run showed clear improvement; in a production setting I'd continue tuning until the documented success criteria were met.
